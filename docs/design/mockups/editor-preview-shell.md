# 编辑与预览页外壳设计稿约束

## 页面目标

第一阶段不重写核心生成逻辑，只统一企业后台外壳、顶部操作区、返回路径和导出任务入口。

## 覆盖页面

- `/project/:projectId/outline`
- `/project/:projectId/detail`
- `/project/:projectId/preview`

## 顶部栏要求

- 返回“我的项目”或“工作台”。
- 显示项目标题和当前阶段。
- 显示保存/生成状态。
- 显示导出任务入口。
- 显示用户菜单和全局配置入口。

## 视觉要求

- 保留编辑区域的工作效率，不增加大面积装饰。
- 工具栏按钮图标优先，必要时配短文本。
- 移动端优先保留核心操作，不让按钮挤压正文。

## 验收

- 大纲生成、描述生成、图片生成、导出逻辑不退化。
- 可编辑 PPTX 导出入口不丢。
- 导出任务失败原因仍能从任务入口查看。
