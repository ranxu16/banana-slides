# 光伏智呈关键界面设计稿索引

本目录用于沉淀前端改造的关键界面设计稿、页面结构说明和验收约束。后续页面实现必须先匹配本目录对应设计稿，再进入编码和截图验收。

## 使用规则

- 每个关键页面必须有一个设计稿说明文件。
- 实现页面前，先阅读对应设计稿说明和 `../guangfu-zhicheng-design.md`。
- 页面实现后，必须用本地环境截图对照设计稿验收。
- 如果设计稿更新，必须同步更新主 DESIGN.md 第 15 节执行记录。
- 设计稿是约束，不是参考氛围；布局、信息层级、关键入口、状态表达都要对齐。
