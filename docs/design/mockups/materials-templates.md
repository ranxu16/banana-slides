# 模板与素材中心设计稿约束

## 页面目标

模板中心和素材中心是一级资源管理能力，不应长期只存在于弹窗或小按钮里。

## 模板中心结构

```text
PageHeader: 模板中心
Toolbar: 搜索 / 场景筛选 / 分组 / 新增模板
TemplateGrid
  TemplateCard: 缩略图 / 名称 / 场景 / 最近使用 / 设为默认 / 使用
```

## 素材中心结构

```text
PageHeader: 素材中心
Toolbar: 搜索 / 类型筛选 / 来源 / 上传素材
MaterialGrid 或 DataTable
  MaterialCard: 缩略图 / 名称 / 类型 / 可见范围 / 使用 / 删除
```

## 关键要求

- 支持光伏业务资产：产品图、逆变器图、项目现场图、品牌图标、常用图表素材。
- 缩略图懒加载。
- 有空状态、上传失败状态、权限不可见状态。
- 素材可见范围要清楚：个人、团队、全局。

## 验收

- `/templates` 和 `/materials` 不再只是 disabled 菜单。
- 可从工作台和编辑页复用素材/模板。
- 资源卡片不使用大圆角和装饰性视觉。
